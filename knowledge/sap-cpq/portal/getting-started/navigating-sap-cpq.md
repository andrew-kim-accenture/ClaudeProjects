Navigating SAP CPQ

The look and feel of your SAP CPQ depends on the Branding and customization. As administrator, you have access to responsive templates so you can change the user interface according to the end user's specific requirements.

This image shows an SAP CPQ environment with the default branding and without any customization.

Note

For end users, SAP CPQ may look different on different tenants. The user interface depends on the quote engine used, the applied customizations, branding, modeling, and so on.

There are two main navigation areas in SAP CPQ: one for the Administrator and one for the End User.
