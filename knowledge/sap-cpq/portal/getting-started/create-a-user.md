Create a User

This procedure explains the minimum requirements to create a user in SAP CPQ.

Click Setup  Users  Users.

Click Add New User.

Enter first and last name of the user.

Enter the email and the username for the user.

Enter the date until which the password will be valid in the Password Expiration Date field.

In User Type, select a user type the user belongs to.

Select a company the user belongs to.

Click Save.

The user is created, and a password creation link is sent to user’s email address.
