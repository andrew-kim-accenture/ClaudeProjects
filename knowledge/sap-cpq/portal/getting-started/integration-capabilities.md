Integration Capabilities

Integration capabilities of SAP CPQ are related to the quote engine in use. Also, some integrations are available out of the box, others are project-based.

When integrated with SAP solutions, SAP CPQ offers a front-office system for managing quotes.

Check the SAP CPQ Integration Guide to see which systems can be used in combination with SAP CPQ. Additionally, to see which integrations are supported in Quote 2.0 and which in Quote 1.0, read Quote 1.0 vs Quote 2.0 - Feature Differences.
