Add to Quote

To add a product from the catalog to the quote, click Add. This button is immediately available for simple products, which don't need to be configured before they're added to the quote. If the Add button isn't available, you need to configure the product before adding it to the quote.

Click Configure.

The configuration screen opens.

Select the options for the product.

Status: Complete appears once the configuration is complete.

Click Add to Quote on the left-hand side.

The configurable product is now added to the quote.
