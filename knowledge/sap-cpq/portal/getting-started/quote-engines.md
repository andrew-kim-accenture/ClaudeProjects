Quote Engines

SAP CPQ traditionally works with the Quote 1.0 engine. The next generation engine, Quote 2.0, was first released in June 2018 and ever since has been developed in parallel.

A long-term plan is for the Quote 2.0 engine to become the default quote engine in SAP CPQ for all clients. Currently, Quote 1.0 is still the default engine.

Quote 2.0 has better performance compared with Quote 1.0 and it's being developed according to latest standards. The features available in Quote 1.0 that are confirmed to be in wide usage, have been improved and replicated in Quote 2.0. Some features have been obsoleted or replaced with other features. Furthermore, the latest integrations are being developed only for Quote 2.0.

For more details on the Quote 2.0 engine, read Quote 2.0 Administrative Documentation.
