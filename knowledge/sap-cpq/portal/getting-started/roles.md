Roles

In SAP CPQ, you can either have the role of an administrator or an end user. Administrators work mainly in the setup part of the system to manage and administer the system for the end users. Access rights and restrictions for administrators are regulated based on user types and permission groups.

The end user role doesn't have access to the Setup. End users configure products and generate quotes exclusively, like for example, sales representative.
