Navigate the Catalog

Products are organized in the catalog into groups called categories. To access a category, select the category image (if available) or the name to display its contents. This displays the products and subcategories belonging to the selected category.

To view product details, select the product name.

Select the Compare checkbox for products you would like to compare. Select the Compare button above the product catalog to display the product comparison page.

Switch between the catalog views by selecting Switch to Tabular/List View on the right-hand side of the screen.

Return to a previous category by selecting the breadcrumbs at the top of the page.
