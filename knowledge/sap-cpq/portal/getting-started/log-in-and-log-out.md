Log In and Log Out

To start to use SAP CPQ, you can either register on the login screen or your administrator can create a user for you.

To log into SAP CPQ, go to the login screen and enter the username, password, and the domain. Select the Remember Me checkbox if you want the system to remember the data for future logins, so you don't need to retype it.

Note

The application parameter Default login remember preference (General   Application Parameters  General Parameters) controls which data the system remembers. The available options are: Remember domain, Remember domain and username, and Automatically login.

Note

To have the domain field automatically prefilled when opening the login page, the link to SAP CPQ needs to contain the domain name. The URL structure is as follows:

app_name/Login.aspx?startdomain=your_domain (for example, yourdomain.cpq.cloud.sap/Login.aspx?startdomain=domainname).

To log out of SAP CPQ, access the menu under the user menu icon in the top-right corner and select Log Out. If you're in the Setup mode, log out by selecting the Log Out icon in the top-right corner.
