Administrator

Administrative side of SAP CPQ is Setup and administrators set up the environment there so end users can best leverage it.

The landing page for the administrator is Setup Home. This page contains the tenant analytics (storage usage and status, overview of user activity).

On the left side of the page, there's a menu with links for accessing different administrative sections.

The following three options are available in the top-right corner of the shell bar at any time:

Help - opens in a new tab the SAP CPQ help documentation with more information about the page you’re currently on.

Exit Setup - redirects you to the SAP CPQ landing page.

Log-out - logs you out of the current SAP CPQ environment.
