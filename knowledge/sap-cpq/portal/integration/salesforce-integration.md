Salesforce Integration

This guide outlines the setup and administration of the integration between Salesforce and SAP CPQ.

This integration is a partner-based solution available only to SAP paper clients, in both quote engines. Two main user journeys are available with this integration: the opportunity-first approach (in which the user creates the new quote in SAP CPQ from the Salesforce opportunity), and the quote-first approach (where the opportunity is created or updated from SAP CPQ).

If you wish to activate the new Salesforce integration in your system, you can download and deploy the integration package relevant for your quote engine and follow the instructions from the installation guide, both of which are available in this knowledge base article. The remaining instructions for setting up the new integration on your system are provided in this guide.
