How to find CPQ API URL

Purpose

You need to have CPQ API URL, which will be used in SAP Cloud Integration system as address path to call the API’s.

Procedure

Login to your SAP CPQ system using an Administrator User.

Click on SETUP in menu bar and choose Setup.

On the top right corner, click on Help icon.

On left navigation, click on CallidusCloud CPQ API.

Check the API Example section.

To access CPQ’s API, use the following URLs:

<application root>/wsAPI/CPQAPI.asmx - this is the set of user level functions like Business Partner Replication and Sales Order Confirmation.

<application root>/wsAPI/wssrv.asmx - this is the set of admin level functions like Material Replication and Pricing Condition Replication.
