System Information

To ensure a smooth and efficient integration of SAP S/4HANA Cloud and SAP CPQ for customer systems, we recommend that you should have already setup the systems listed below, before starting the integration process:

SAP S/4HANA Cloud (2302)

SAP CPQ (2211 or higher)

SAP Business Technology Platform (SAP BTP, Cloud Foundry Environment)

SAP Cloud Integration with initial configuration

SAP Variant Configuration and Pricing with initial configuration

SAP Master Data Integration with initial configuration

SAP Event Mesh with initial configuration

Note

For integration of SAP Variant Configuration and Pricing, please execute the steps that are mentioned in Setting Up Set Up Configurable Model using Variant Configuration (22T).
