Integrate SAP CPQ Quote 1.0 with SAP Commerce Cloud

Complete this procedure to enable the SAP Commerce Cloud integration on the SAP CPQ side.

Procedure
Go to Setup   Providers   Providers   SAP.

Available SAP providers display.

Click SAP Cloud Platform.

A new page displays.

On the Commerce Settings tab, enable the Connect SAP Commerce toggle switch.
From the Commerce Notification dropdown list, select the previously created RFQ notification as explained in Prerequisites for SAP Commerce Cloud Integration for Quote 1.0.
In Release Quote Endpoint, enter the endpoint that sends the quote back to SAP Commerce Cloud.
In Release Quote And Proposal Endpoint, enter the endpoint that sends the quote together with a proposal to SAP Commerce Cloud.
Click Save.
