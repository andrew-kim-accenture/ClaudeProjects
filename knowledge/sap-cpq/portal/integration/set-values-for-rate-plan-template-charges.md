Set Values for Rate Plan Template Charges
Context
Note

This integration is deprecated.

Procedure
In Parameters, define the Initial Value for every parameter.

Parameters are predefined as part of the rate plan template in SAP Subscription Billing. The currency for parameter values is defined under Currency in the Extended Rate Plan section.

(Optional) In the Charges section, you can see a detailed overview of the charges, updated with the specified parameter values.

It is possible to set the fixed amount for a charge without using a parameter, but that can only be done on the SAP Subscription Billing side.

If all the steps from Create SAP Subscription Billing Product are complete, click Save to create the SAP Subscription Billing product.

The product is now added to the catalog.
