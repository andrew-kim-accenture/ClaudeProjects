SAP Sales Cloud - SAP CPQ Integration for Quote 1.0

SAP CPQ can be integrated with SAP Sales Cloud providing a streamlined workflow for users managing their sales quotes in SAP Sales Cloud.

The following pages describe the procedure to configure the Basic Integration between SAP Sales Cloud and SAP CPQ using SAP Cloud Integration. The instructions are focused on the users who will be managing the integration in a new integration tenant, not in an existing tenant. Users with an existing tenant will also find the instructions useful, although they may come across errors that are bound to the existing modeling on their tenants.

In SAP Sales Cloud, the SAP CPQ quote serves as the only quote, with the SAP Sales Cloud sales quote generated in the background. Once you access SAP CPQ for the first time, the sales quote created in SAP Sales Cloud becomes non-editable. Opening it redirects you to the quote in SAP CPQ. Additionally, the user journey in SAP Sales Cloud starts on the sales quote, not the opportunity.

Network Security team takes the responsibility of preparing the network environment across different systems and its security aspects.

Configurable products synchronized from SAP Variant Configuration and Pricing to SAP Sales Cloud can be used in a scenario for the SAP CPQ integration.
