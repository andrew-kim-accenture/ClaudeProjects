Opportunity Status

This section defines the mappings between a CRM opportunity status and a SAP CPQ quote status. These mappings are only done when the Create/Update Opportunity action is initiated, either by the user clicking the action or by a pre or post action.

To create a new mapping, click Add New. On the opportunity mapping definition page, choose the CPQ Cart Status from the list box. To choose the CRM Opportunity Status, click CRM Lookup. This displays all the opportunity statuses from the CRM application.

Note

The integration is case sensitive. Use CRM Lookup to select CRM Fields.

When a status is selected, it is automatically entered into the CRM Opportunity Status field.
