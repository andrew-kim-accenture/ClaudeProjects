CX AI Integration

SAP CPQ can use artificial intelligence to recommend cross-sell products and discounts on quotes.

To enable recommendations, the SAP CPQ tenant must be integrated with CX AI that pulls data from the system, analyzes it and returns recommendations to SAP CPQ.

Note

CX AI integration works with both Quote 1.0 and Quote 2.0.



Set up CX AI Integration

All CX AI integration settings are in the Providers section.
