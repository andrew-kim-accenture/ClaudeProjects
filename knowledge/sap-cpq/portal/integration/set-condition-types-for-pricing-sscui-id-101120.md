Set Condition Types for Pricing (SSCUI ID: 101120)

Follow the below steps to create a new Condition Type:

Test Step #

	

Test Step Name

	

Instruction

	

Expected Result

	

Pass/Fail/Comment




1

	

Log On

	

Log on to the SAP Fiori Launchpad with your user credentials as Configuration Expert - Business Process Configuration role assigned.

	

The Fiori Launchpad is displayed.

	 


2

	

Access the Manage Your Solution App

	

Open Manage Your Solution and then choose Configure Your Solution.

	

The Manage Your Solution screen is displayed.

	 


3

	

Access App Make Specifications

	

Search with following criteria:

Application Area: Sales

Sub Application Area: Price Management

Choose the group Set Condition Types for Pricing

	

The "Maintain condition types": Change screen is displayed.

	 


4

	

Maintain Condition type

	

Choose New Entries

Maintain the following for the Z condition type and a description

Condition Class: A

Calculation Type: A

Under, Changes which Can be Made, make the following entries:

Amount/Percent: X

Header Condition: X

Item Condition: X

Delete: X

Choose Save

	

New Z Condition type is created
