Integration with Central Entry Point

You can access SAP CPQ directly from SAP S/4HANA by using the Central Entry Point. The Central Entry Point provides a single entry point to SAP applications and content. SAP CPQ is available as a tile that launches the application directly.

This integration means you can start SAP CPQ without navigating to a separate URL or application. SAP CPQ becomes part of the SAP S/4HANA user experience and can be accessed in the same way as other SAP applications.
