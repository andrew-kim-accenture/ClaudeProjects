Create SAP CPQ Communication User

A communication user need to be created in SAP CPQ System, which will be used in SAP Cloud Integration for basic authentication to SAP CPQ.

Login to your SAP CPQ system using an administrator user.

Click on Setup in menu bar and choose Setup.

In left navigation bar, select Users under Users.

Click Add New User.

Enter the First Name, Last Name and Email Address for Communication User.

Enter Username as COM_USER_CAL and maintain the Initial Password.

Choose User Type as Sales.

Select Administrator.

Select Company as SAP.

Click Save.
