Integration Overview and Prerequisites

This is a high-level overview of all the prerequisites and steps that need to be performed in order to integrate SAP CPQ Quote 2.0 and SAP Sales Cloud V2.

The SAP CPQ instance with the Quote 2.0 engine enabled is in place.

SAP CPQ and SAP Sales Cloud V2 need to be synchronized in terms of product (material) master, involved parties, markets and pricebooks via the SAP S/4HANA.

The SAP Sales Cloud V2 tenant and initial user details are in place.

The SAP Cloud Integration tenant is set up.

A a matching user is created in both SAP CPQ and SAP Sales Cloud V2.
