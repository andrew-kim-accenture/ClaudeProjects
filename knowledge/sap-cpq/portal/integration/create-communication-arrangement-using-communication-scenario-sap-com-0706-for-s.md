Create Communication Arrangement Using Communication Scenario SAP_COM_0706 for Solution Order

Purpose

This section describes how to set up communication scenario SAP_COM_0706 for transactional data integration for solution orders by using the Communication Arrangement app.

Procedure

Log on to the SAP Fiori launchpad as an administrator.

Open the Communication Arrangements app.

To create a new communication arrangement, choose New.

In the New Communication Arrangement dialog box, enter the following data.

Field

	

Value




Scenario

	
SAP_COM_0706 (Business Solution Order Integration)
Tip

You can use the Value Help to search for a specific scenario.




Arrangement Name

	

Enter the communication arrangement name, for example, <SAP_COM_0706_SCI_SolutionOrder>

Choose Create.

Maintain Common Data.

In the Communication System, select the communication system for the SAP Cloud Integration connections created in the Create Communication System topic. For example, <SCI_123>.

The Inbound Communication user is entered automatically.

Choose Save.

The communication arrangement is now active.
