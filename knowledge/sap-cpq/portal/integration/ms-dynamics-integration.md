MS Dynamics Integration

SAP CPQ can be integrated with the MS Dynamics CRM solution.

Note

All mapping of entities in the two systems should be handled via IronPython scripting, and not in the SAP CPQ Setup.

In addition, if you are using Quote 2.0, you need to develop the integration using the toolset for developing custom CRM integrations instead of using the options described in this guide.
