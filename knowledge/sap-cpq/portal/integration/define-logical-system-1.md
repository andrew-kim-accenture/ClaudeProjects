Define Logical System

Purpose
This describes the procedure to create a logical system that represents the SAP CPQ system in SAP ERP system. Logical systems are defined as cross-client systems.
Procedure

Access the following transaction:

Transaction Code

	

BD54

Choose New Entries.

Create a logical system that identifies the on-demand system, as given in the table below:

Field

	 


Logical System*

	

<logical system name of SAP CPQ tenant> Ex: CPQINTERP




Name

	

<Description> for example: SAP CPQ.

Choose Save.

Choose Ok to confirm the pop-up.
