Maintain Pricing Procedure in SAP S/4HANA Cloud

To enable integration between SAP CPQ and SAP S/4HANA Cloud, adjustments are needed in the standard pricing procedure.

Discounts are processed in a certain way in SAP CPQ that makes it necessary to change the standard pricing procedure. Please adopt the changes that are mentioned below before you start creating a quote in SAP CPQ.



Set Condition Types for Pricing (SSCUI ID: 101120)

Assignment of Pricing Procedure (SSCUI ID: 101117)

Set Customer Pricing Procedures (SSCUI ID: 101202)

Set Pricing Procedure Determination (SSCUI ID: 101118)
