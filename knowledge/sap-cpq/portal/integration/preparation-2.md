Required Information
We recommend that the consultant performing the implementation has the following implementation and configuration knowledge:

SAP ERP

SAP Cloud Integration

SAP CPQ

SAP Variant Configuration and Pricing (optional)

Prerequisites
Before you set up SAP CPQ Integration with SAP ERP, ensure that you have:

SAP CPQ instance.

Ensure that your SAP CPQ instance is in place before you start implementing SAP CPQ integration with SAP ERP.

SAP ERP System.

Ensure that your SAP ERP system, support package component CODERINT is at SP-Level 43.

SAP Cloud Integration tenant.

The SAP Cloud Integration Runtime URL is included in the SAP Cloud Integration provisioning e-mail. Example: https://<tenant>-iflmap.hana.ondemand.com.

SAP Variant Configuration and Pricing instance (optional)

Ensure that your SAP Variant Configuration and Pricing instance is integrated to SAP ERP system and your configured products are replicated between SAP ERP and SAP Variant Configuration and Pricing.
