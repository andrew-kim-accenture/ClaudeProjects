Set Sold-To Party to Mandatory

Procedure:

Login to SAP CPQ system using an Administrator user.

In the left menu bar, choose Setup  Setup.

In left navigation bar, select Workflow/Approval  Required Fields by Workflow Action.

The table shows field names on the vertical axis and actions on the horizontal axis.

Check the intersection of Sold-to Party and Place Order.

Choose Save.

As the Sold-To Party is mandatory, it is useful to display this partner function by default for a new quote.

Login to SAP CPQ system using an Administrator User.

In the left menu bar, choose Setup  Setup.

In left navigation bar, select Quotes  Partner Functions.

Choose Edit for Sold-to party function.

Turn on the Show by default on new quotes.

Choose Save.
