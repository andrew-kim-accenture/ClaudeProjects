Create Measurement Attributes
Prerequisites

Add a unit of measurement which is going to be associated with the values you add for this attribute.

Complete the common attribute creation procedure.

Procedure
Select Measurement in the Attribute Type dropdown.
In the Unit of Measurement dropdown, select the one unit for which you wish to add the attribute values.
Click the Add Values button and select the number of rows you wish to use.

A table displays.

Enter the Value and Value Code for each row.

The Value System ID will be generated automatically, according to the value you’ve entered.

Click Save.
