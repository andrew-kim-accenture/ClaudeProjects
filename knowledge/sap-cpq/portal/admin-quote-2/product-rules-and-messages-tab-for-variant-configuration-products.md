Product Rules and Messages Tab for Variant Configuration Products

Visibility Restrictions and Product Messages are displayed under the Rules and Message stab.

Under the Visibility Restrictions tab, rules can be defined to limit the possibility of configuring the product. This ensures that the user is only able to create an accurate and valid configuration of the product based on the rules established in Setup. The Rules and Messages tab allows for the creation of complex rules using SAP CPQ tags. The Formula Builder, which can be accessed during the creation of every rule, can help create these rules in a user-friendly way.
