Defining Key Values

You can easily change values of custom dictionaries in General  Dictionaries.

Click  next to the dictionary you wish to update.

Use the search fields to filter dictionaries.

Click Add new key.

Define the key and its value.

Please note that for an optimized usage of dictionaries, you should only copy/paste the keys whose values you wish to override in the existing dictionaries, not define new ones.

Click Save.

The key and its value are added to the list from which you can always delete and edit them. Dedicated columns show you when each key was created and when it was last modified.
