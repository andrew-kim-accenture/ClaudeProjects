Audit Trail Deletion

It's possible to define time periods after which audit trail data is deleted.

On the Audit Trail Deletion tab, you define a time period for:

Automatic deletion of administrator actions, user actions, and changes on products, attributes, pricebooks and custom tables from their respective audit trail logs.

Automatic deletion of personal data log records.



Set Up Automatic Deletion of Data from the Audit Trail
