Set Up Automatic Deletion of Personal Data Log

You can set up automatic deletion of the Personal Data Log.

Context

All Personal Data Log entries are by default automatically deleted after six months, or less, if they’re no longer required by the company that holds that information. The default retention period of personally identifiable information can be changed if needed.

Procedure
Go to Setup  Security  Data Deletion.

A new page displays.

In Audit Trail Deletion  Personal Data Log, enable the Delete Data Protection Log toggle switch.

New fields display for defining the period after which the data will be deleted.

Select Month or Years and enter the number in the first field.
Click Save.

A confirmation message displays.
