Create Change Set

This topic lists the two ways to create change sets.

You can access the Change Set dropdown in the Setup. The names and IDs of active change sets in your environment are displayed here.

Click  next to the Change Set dropdown in Setup.

A popup window displays.

Enter the name of a new change set in the indicated field.

(Optional) Enter the description of a new change set in the indicated field.

Click Save.

Once you create this change set, it’s automatically activated.

Alternatively, you can:

Go to Setup   Audit Trail  Change Set.

The grid containing all available change sets (except the default one) displays. It has the following columns: Edit, Active, Name, Description, Change Set ID, Date, and User.

Click New Change Set.

A popup window displays.

Enter the name of a new change set in the indicated field.

(Optional) Enter the description of a new change set in the indicated field.

Click Save.

(Optional) To edit column visibility, click .

The new change set is now listed in the Change Set grid. Once created, the change set can be edited.

Note

Currently, change sets can’t be deleted, but they can be deactivated with the Active toggle switch.
