Distribution Channels

The Distribution Channels administrative section lets you view replicated distribution channels from SAP S/4HANA.

A distribution channel defines how products or services reach customers. Examples include retail, online, e-commerce, and direct sales.

Each distribution channel in a table includes the following:

Field

	

Description




Name

	

The name of a distribution channel as defined in SAP S/4HANA.




System ID

	

Automatically generated from the distribution channel's code by adding _cpq.




Code

	

The distribution channel's code in SAP S/4HANA.




Is Default

	

Sets the distribution channel as the default one when selected. The name of the distribution channel for which the flag is set to TRUE appears in the quote under the Distribution Channel field as the preselected option from the dropdown. Therefore, there can only be one default distribution channel.

Available Actions

You can modify or delete the replicated data or add a new distribution channel. Although all these actions are supported, this page is typically used just for viewing the replicated data.
