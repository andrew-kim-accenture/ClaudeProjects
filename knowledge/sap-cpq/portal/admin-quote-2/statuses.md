Statuses

The Statuses link under Workflow/Approval drop-down provides information about an order or quotation status to users.

Common and Additional Statuses

SAP CPQ allows you to define the order or quote statuses as per your needs.

Examples of common statuses are Open, Order Placed and Shipped.

Additional status definitions can also be added. Some examples are listed below:

For quotation tracking, you can add the following statuses: Open Quote, Closed Quote, Converted to Order.

For orders, you can create the following statuses: In Engineering, On Hold, or At Assembly Floor.

Default Status

To set a status as the default, click the edit  button next to the status and enable the Default option.

If another status is already set as the default, you will receive a confirmation message. This message informs you that a new status is about to be set as the default. Click OK to make the current status the default. Click Cancel to keep the existing default status unchanged.

Remember: there can't be two default statuses, and there must always be one status set as the default. If you try to mark the default status as non-default, you'll see an error message.
