Incoterms

The Incoterms administrative section lets you define incoterms for use in the quotation process. Since incoterms aren't replicated from SAP S/4HANA, you need to maintain them manually in SAP CPQ.

Each incoterms row in a table includes the following:

Field

	

Description




Name

	

The name of the incoterms as defined in SAP S/4HANA.




System ID

	

Automatically generated from the incoterms' code by adding _cpq.




Code

	

The incoterms' code in SAP S/4HANA.

Available Actions

You can add new incoterms, or modify or delete existing ones.
