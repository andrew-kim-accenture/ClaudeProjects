Quote Fields

In the Quote Fields section in Setup  Quotes administrators can set and manage Standard Quote Fields and Custom Quote Fields.

In the Quote Standard Fields tab, you can view Standard Quote Fields, set permission priorities, and define editability and visibility permissions based on Quote status and user permission group. Learn more on the Quote Standard Fields page.

In the Quote Custom Fields tab, you can create custom fields in the Quote, define permissions based on the user and the status of the quote. Learn more on the Quote Custom Fields page.
