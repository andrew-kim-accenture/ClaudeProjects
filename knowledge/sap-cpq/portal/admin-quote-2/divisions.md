Divisions

The Divisions administrative section lets you view replicated divisions from SAP S/4HANA.

A division is a grouping of broad categories of products in one organizational unit. For every division, you can make customer-specific agreements on, for example, partial deliveries, pricing, or payment terms.

Each division in a table includes the following:

Field

	

Description




Name

	

The name of a division as defined in SAP S/4HANA.




System ID

	

Automatically generated from the division's code by adding _cpq.




Code

	

The division's code in SAP S/4HANA.




Is Default

	

Sets the division as the default one when selected. The name of the division for which the flag is set to TRUE appears in the quote under the Division field as the preselected option from the dropdown. Therefore, there can only be one default division.

Available Actions

You can modify or delete the replicated data or add a new division. Although all these actions are supported, this page is typically used just for viewing the replicated data.
