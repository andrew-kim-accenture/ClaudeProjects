Bulk Quote Deletion

The Bulk Deletion section allows administrators to delete quotes.

Administrators can clean up the system database by deleting old quotes. You do this under Quotes  Bulk Deletion.

Procedure

Search for the quotes in the table.

Select with the checkbox the quotes that you want to delete. You can also select the all checkbox to select all the quotes within the table (with or without filters).

Choose Delete.
