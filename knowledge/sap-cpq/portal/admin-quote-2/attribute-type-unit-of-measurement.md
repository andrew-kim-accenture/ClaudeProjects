Attribute Type - Unit of Measurement

Measurement attributes need to be created in the system to support Unit of Measurement in product configuration.

You can create a new attribute type (Setup  Product Catalog  Attributes). As soon as Measurement is chosen as an attribute type, a Unit of Measurement field is displayed. Here, you can select one of the measurement types you've previously added to the system. This field is required.

At this point, you can add attribute values and attribute value codes, in the same way you do for other attribute types; and add the attribute to the product.
