Section Distribution

A quote section can be distributed into other sections in the Solution Design tab.

When the distribution is initiated, a new distributed section is created in each section to which the source is being distributed. The value of the distributed section is split between the target sections as indicated in the distribution method.

To be able to distribute a section to other sections, you first need to enable the distribution for it either in the Section Builder or the excel file when creating/editing a section. Consequently, the button  displays in the Solution Design tab, next to the section that can be distributed.
