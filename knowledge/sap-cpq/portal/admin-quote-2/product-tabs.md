Product Tabs

Use product tabs to organize information on the product configurator.

Creating a Product Tab
Tabs are invisible until you use them in product configuration. To create a tab:

Go to Setup  Product Catalog  Tabs.

A page with all existing tabs is displayed.

Click Add New.

Enter a name of the tab in Tab name.

Tab system Id is automatically created based on the tab name you entered.

Enter the rank in Default tab rank.

The defined rank is applied to the tabs in the configurator and it determines the order in which they are displayed. The default rank will also be applied to custom quote tabs until you override it with a more specific rank.

(Optional) Select Show tab header in configurator.

If selected, the tab header is displayed in the product configurator.

Click Save.

The page with all existing tabs is displayed.

After you create a tab, you can use it for product configuration.
