Unit of Measurement on User Page

Units of measurement can also be accessed from User Menu  User Page.

As soon as a new measurement type is added to the system, that measurement type becomes a standard field on the User Page and User Setup.

The Unit of Measurement field labels can be translated to the user’s language by adding its key to the dictionary. The content of these fields, however, can’t be translated. You can add a default value to the Unit of Measurement fields in User Setup. A default value can't be added through import, Users functionality, or user API.
