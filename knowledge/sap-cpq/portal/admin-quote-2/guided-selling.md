Guided Selling

Guided Selling is an automated Catalog filtering process. It helps sales representatives easily find products that match their needs by narrowing down the products in a category based on their attribute value selections.

When a category contains products that share multiple attributes, you can create guided selling. Sales representatives can activate it if they need assistance while browsing the Catalog. For example, the Cars category consists of multiple products with attributes such as year of manufacture, body style, and speed. You can list the values of these attributes in a guided selling, allowing sales representatives to select the ones they are interested in browsing. If sales representatives want to find all minivans manufactured between 2000 and 2009, with a top speed of at least 70 mph, they can do so more quickly using this method.

The guided selling feature is available to sales representatives through the Guided Selling button that is displayed in the top-right corner of a category for which it is available. When a user clicks the button, a context menu displays the available product attributes for filtering.



Create Guided Selling

Guided selling can be either simple or complex, depending on the use case. You can only add attributes, or you can create rules and scripts for more complex business scenarios.
