Translate Partner Function

The name of a partner function can be translated to any of the system languages.

Procedure
In Setup, go to Quotes  Partner Functions.
Click  next to the partner function you want to translate.

A submenu displays.

Click Translate this item.

The Translate Partner Function page displays.

The Partner Function Information contains the name of the partner function as defined in Setup, and the partner function value in the default language.

Click Add New Translation.
Find the target dictionary in Dictionary.
Enter the translation in Translation value.
Click Save.

The name of the partner function is translated to the selected language.

Next Steps

When the target language is selected as user’s language, the translated value displays in quotes. In the list of partner functions in Setup, the name of the partner function always displays how it was defined, not translated.
