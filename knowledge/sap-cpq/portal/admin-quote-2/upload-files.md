Upload Files

Before you initiate the upload, click the destination folder to which you wish to upload the file in the left-hand navigation pane.

Procedure
Click Upload.
The Upload Files window pops up.
Note

The maximum size of the file or selected group of files being uploaded is 20MB per file. For ZIP files, the maximum size of the ZIP file is 20MB, not per file within the ZIP.

Click Add….

A window for browsing your local computer displays.

Select the file for upload and click Open.

You can repeat this step to select multiple files. All files are listed in the Upload Files window.

Click Upload.

The progress bar visually shows the time remaining for upload.

The file is uploaded to the selected folder.
