File Import

This topic contains the procedure for importing units of measurement into SAP CPQ.

Procedure
Click the Edit icon of your Unit of Measurement.
Click Browse to choose your file.
Click Save.

The import is completed automatically once you upload the file. Creating measurement types and units has to be completed prior to creating the attribute of the Unit of Measurement type.
