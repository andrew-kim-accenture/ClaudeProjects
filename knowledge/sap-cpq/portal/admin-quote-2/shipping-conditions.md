Shipping Conditions

The Shipping Conditions administrative section lets you define shipping conditions for use in the quotation process. Since shipping conditions aren't replicated from SAP S/4HANA, you need to maintain them manually in SAP CPQ.

Each shipping condition row in a table includes the following:

Field

	

Description




Name

	

The name of the shipping conditions as defined in SAP S/4HANA.




System ID

	

Automatically generated from the shipping conditions' code by adding _cpq.




Code

	

The shipping conditions code in SAP S/4HANA.

Available Actions

You can add new shipping conditions or modify or delete existing ones.
