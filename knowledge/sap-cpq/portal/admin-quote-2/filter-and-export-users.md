Filter and Export Users

The filtering option in Bulk User Import/Export allows you to export only the users that match your search criteria.

Context

Procedure
In Users, click Filter and Export and a pop-up displays.
Select the field per which the users are filtered.
Select an operator and enter a value.
Repeat the steps to add filters.
Click Export.

A pop-up message informs you that the export has started and a job entry displays in the list.

Click Finished (Download) in the Status column to download the export file.

The system has filtered the users per your search criteria.

The file contains all their details except the passwords.

The maximum number of parameter limit in bulk user export job is 2000. An error message displays when the maximum parameter limit exceeds.
