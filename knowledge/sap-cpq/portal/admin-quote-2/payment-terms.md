Payment Terms

The Payment Terms administrative section lets you define payment terms for use in the quotation process. Since payment terms aren't replicated from SAP S/4HANA, you need to maintain them manually in SAP CPQ.

Each payment term row in a table includes the following:

Field

	

Description




Name

	

The name of the payment terms as defined in SAP S/4HANA.




System ID

	

Automatically generated from the payment terms' code by adding _cpq.




Code

	

The payment terms' code in SAP S/4HANA.

Available Actions

You can add new payment terms or modify or delete existing ones.
