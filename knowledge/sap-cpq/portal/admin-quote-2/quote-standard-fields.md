Quote Standard Fields

Listed here are the standard quote fields that are present in the quote header by default.

Note

This topic does not include all required fields. Refer to the SAP CPQ Scripting Documentation to access the complete list of fields.

Translations for standard quote fields are available in the dictionary in all supported languages. Learn more at the page: Dictionaries.

Standard Quote Fields

You can see the following standard quote fields in the quote header when Sales Area isn't activated.

Standard Quote Field

	

Description




Status

	

The current status of the quote in the workflow (such as Open, Preparing, Approved, Rejected, Order Placed, and so on).




Market

	

The market selected for the currently open quote, with a defined currency, market factor, and pricebooks.




Pricebook

	

The pricebook selected for the currently open quote, which contains defined product prices.




Effective Date

	

The date that determines which prices are displayed for items in the quote, depending on the period for which the selected pricebook entries are valid.

The list of the standard quote fields that user can set when the Sales Area is activated:

Standard Quote Field

	

Description




Sales Organization

	

The name of a sales organization.




Distribution Channel

	

The way products or services reach customers. Examples include retail, online, e-commerce, and direct sales.




Division

	

A grouping of broad categories of products in one organizational unit.




Sales Office

	

The sales office to which the business transaction is assigned




Sales Group

	

The sales group in SAP CRM to which the business transaction is assigned.




Payment Terms

	

Payment terms for use in the quotation process.




Incoterms

	

Incoterms that should be used in the quotation process.




Shipping Condition

	

The name of the shipping conditions.




Status

	

The current status of the quote in the workflow (such as Open, Preparing, Approved, Rejected, Order Placed, and so on).




Effective Date

	

The date that determines which prices are displayed for items in the quote, depending on the period for which the selected pricebook entries are valid.
