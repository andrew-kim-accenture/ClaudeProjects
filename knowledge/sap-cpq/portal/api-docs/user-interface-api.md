User Interface API

User Interface API is intended for SAP CPQ to use to perform certain operations, such as retrieving user types.

It is not recommended for end users, since all operations performed with this API require several iterations to complete. In other words, they aren’t very user-friendly unlike SAP Standard REST APIs.

All APIs which contain the /rd/ element in their endpoints are User Interface APIs.
