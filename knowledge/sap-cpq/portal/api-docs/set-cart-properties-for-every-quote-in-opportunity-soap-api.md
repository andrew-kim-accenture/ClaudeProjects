Set Cart Properties For Every Quote In Opportunity SOAP API

SetCartPropertiesForEveryQuoteInOpportunity performs setting/updating of custom fields for every quote in the opportunity by using setCartProperties web method.

API parameter username identifies user

API parameters sessionId and url are received from SalesForce

API parameter domain identifies domain under which the quotes are created

API parameter opportunityId identifies opportunity where the quotes belong to
