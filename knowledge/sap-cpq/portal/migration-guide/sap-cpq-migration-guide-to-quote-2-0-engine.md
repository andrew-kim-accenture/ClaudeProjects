SAP CPQ Migration Guide to Quote 2.0 Engine

This document describes recommended migration path for clients who decided to migrate their Quote 1.0 model to the Quote 2.0 model and need guidance on the required steps. Migration to the Quote 2.0 engine needs to be handled as a project, and the goal of this document is to provide needed information about each step in order to ensure smooth transition to new engine.

This guide will help you understand the following:

Benefits of using Quote 2.0

Quote 1.0 vs Quote 2.0 Feature Differences

How to migrate from Quote 1.0 to Quote 2.0

How to estimate the Effort needed for Migration

Additionally, in this document, you can find a comparative overview of the features available in Quote 1.0 and in Quote 2.0, as well as the specifics relevant for the migration of each administrative section.
