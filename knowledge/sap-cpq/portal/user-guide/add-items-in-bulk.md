Add Items in Bulk

Add Items in Bulk allows you to search multiple part numbers by entering or copying them from another document, such as a Microsoft Excel file.

Note

If administrators wish to enable adding items in bulk for their underlying users, they can activate it in the setup under General  Application Parameters  Shopping Cart and Quotes  Enable Adding Items in Bulk. This application parameter is supported for both quote engines.

Apart from being able to search product descriptions, adding items in bulk functions similarly to the quick search. Unlike the quick search, though, adding teams in bulk allows you to search for multiple part numbers and immediately add products to cart, along with the desired quantities. To access this feature, simply click Add Items in Bulk at the top of the Quote List page. Alternatively, you can click Add Items in Bulk directly on the quote.

The page contains an area where you can enter:

Part Numbers Only

Part Number and Quantity

Quantity and Part Number

If you want to enter quantities along with part numbers, you can choose to enter part numbers first, and quantities second, or the reverse order – quantities first. If you want only to enter part numbers, leave Part Numbers Only checked.
Note

The page is designed to work well when copying part numbers and quantities from Microsoft Excel. When copied from Microsoft Excel, every part number/quantity combination is copied into each single row, so the number of rows in Microsoft Excel corresponds to the number of rows in the text area. Part number and quantity are delimited by the Tab character. It's also possible to enter part numbers and quantities manually, but the user must ensure to enter only one part number or quantity combination into every row of the text area. Additionally, the user can delimit part numbers and quantities with a blank (space) character.

When the system finishes searching for the entered part numbers, it will redirect you to a new page, which displays results in a tabular view. There, you can see given part numbers, associated product names, their list prices and text boxes that can be used to enter quantities, and to add offered products to the quote. If quantities were entered along with part numbers, then the quantity textboxes are prepopulated.

If you want to add more products to the list, click Back and add new part numbers. Numbers previously entered are prepopulated, so add only the new part numbers.

Once you have specified all the necessary products, click Add to Quote. If there is a currently active quote, the products will be added to that quote. Otherwise, a new quote is created and the products are added to it.

Note

Variant Configuration items can't be added to the quote in bulk, in either of the quote engines.
