Previewing the Template

Depending on your assigned role, you may not have the authorization to generate documents prior to the quote being approved. In this case. you can preview the documents in the document generation process from both the Choose Template page or the Customize page.

Note

It's not possible to preview .zip, excel, or .txt file types.

The document preview is only generated in PDF format regardless of the chosen document generation type.

When you click on the preview icon () on the Choose Template or Customize page, the document preview opens in a preview window. From the document preview you can:

Use the up or down arrows to navigate to each page of the document.

Use the zoom function to zoom in or out on the document pages.

View the total number of pages of the document to be generated.

Download the preview document.

Tip

You can also preview additional content and additional document files that have the file types, .jpg, .jpeg, .gif, and .png. For these file types, the preview and file download will be in the original format (not in PDF).

Note

Each page of the preview document is watermarked with Preview, both on the screen and when downloaded for security purposes. For is a user doesn't have authorization to generate documents before a quote is approved.
