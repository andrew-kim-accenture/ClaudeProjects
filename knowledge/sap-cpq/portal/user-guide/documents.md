Documents

The Documents tab allows uploading of files to the quote. This feature is useful for inclusion of any supporting documents to the quote.

Any file can be uploaded and, once uploaded, it’s possible to Download and View each file. To upload a file, you can either click Add or drag and drop the file.

You can upload and delete multiple documents using REST APIs.
Note

An error message is displayed if the documents do not comply with the SAP CPQ validations.

The application parameter Allowed Content File Extensions defines which file formats are supported. The files can also be deleted from the quote.

Attached files can be emailed as attachments after the quote generation process. Attachments are useful for sending files to users who would receive a generated quote.

Note

In the Documents tab of a quote that has been copied or derived from another quote, it is possible for the Date Created of those files to be older than the creation date of the copied quote. This is because the Documents tab retains all additional files and their original upload dates from the initial quote.
