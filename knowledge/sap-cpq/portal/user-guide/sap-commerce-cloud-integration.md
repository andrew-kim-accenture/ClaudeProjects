SAP Commerce Cloud Integration

The SAP CPQ – SAP Commerce Cloud integration is supported for both SAP CPQ Quote 1.0 and SAP CPQ Quote 2.0 and can help increase front office sales efficiency and enables B2B customers to directly interact with sales representatives via collaboration functionalities that both systems have.

When you create a quote and navigate to either Generate Documents or Generate Documents for E-Sign, on the Download Document and Send Email page you can choose to send the quote to SAP Commerce Cloud.

To send a quote to SAP Commerce Cloud, follow the same process for either Generate Documents or Generate Documents for E-Sign. After selecting Generate and adding all the information on the Download Document and Send Email page, select Send to Commerce from the available integrations on the page footer. If Send to Commerce is the only available integration, you can use the Send to Commerce button. If more than one integration is available, the first available integration is shown as a button while other interations are available in the 3 dot menu next to the first integration button. In the Send to Commerce window, add the message to be sent to SAP Commerce Cloud and select Submit to Customer. If the quote is successfully released to SAP Commerce Cloud, a success message will be displayed at the top of the page. If you receive the error message Release Quote And Proposal To SAP Commerce action is not available in this status, check that the status of the quote is set to Preparing, you cannot release a quote to SAP Commerce Cloud in the Open status.

Note

To release a quote to SAP Commerce Cloud, you must have the correct user role, if the user role isn't assigned you'll receive and error message directing you to contact your administrator.
