Parallel Work on Quote 1.0 (User Side)

Quote 1.0 does not support parallel work on quotes. This means that only one user at a time can edit a quote. Although the system won’t disable editing if there are multiple users, you should avoid making any changes as that will cause errors and wrong values.

You will be informed that other users have recently opened the quote you are working on via an alert in the top of the quote. That is the default format of the alert, but it can also be displayed in a pop-up after opening a quote.
