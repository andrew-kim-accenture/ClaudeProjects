Images for Quote Actions

Administrators can enable images or icons for quote actions so that users can easily spot them.

Setup
Activation

Under Workflow/Approval  Actions, administrators must activate Display images for quote actions. By default, this setting is deactivated.

Display in Quotes

Administrators can edit the following properies of an action with an image:

Rank: Sort order within quote.

Actions appear following the defined rank or sort order.

Actions with an empty rank field appear at the end.

Primary Action: Enable it if you want that the action is always displayed to users. A primary action can't be collapsed.

Icon: Upload the image for quote action. The supported image size is 32x32 pixels.
