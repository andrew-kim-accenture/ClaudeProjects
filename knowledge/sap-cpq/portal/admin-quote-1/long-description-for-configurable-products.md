Long Description for Configurable Products

Long descriptions for configurable products enable the display of more details of attribute values.

Administrators can define a long description for the attribute values of configurable products. This allows administrators to display more information about an attribute value in the Configurator and Quote.

By default, this feature is turned OFF (position of the switch is FALSE). To enable it an application, switch the parameter Display attribute value descriptions (Configurator and Quote) to TRUE.

It is possible to have an Attribute description on a Global and on a Product attribute definition level.

When the Global attribute is assigned to the product, you can redefine an attribute value description for that specific product and override the global description. For more information, see Product Attributes Tab.

The following conditions apply to the long descriptions:

Language support is available.

Formulas are not supported.

For variant priced products, there is no limitation of the character numbers.

The user can display the attribute value description by hovering the mouse pointer over the attribute value.
