Developer Console Groups

The Groups section lists the main traces (high-level traces - Trace H) so you can expand them and have a hierarchical overview of the underlying traces.

You can filter groups by entering a keyword. The filter can also be negative (for example, you can choose to view all the traced lines that don’t contain a particular word, such as clone, for example). The feature also enables filtering support for characters, asterisks, and question marks. The question mark replaces one character and the asterisk replaces any character and any number of characters.
