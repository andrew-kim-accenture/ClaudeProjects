Cart Fields Administration

The Cart Fields administrative section allows you to define which fields in the cart will be editable for each user group and order status.

It is also possible to define which calculations will be executed on action calculate for each Editable Field Group. An Editable Field Group is a set of cart fields, which are associated with the same calculations. Actions performed in Cart Fields Administration are logged in Audit Trail  Admin Actions.
