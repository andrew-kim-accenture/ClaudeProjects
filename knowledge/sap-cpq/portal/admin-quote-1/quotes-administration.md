Quotes Administration

This section contains description of all features that are related to creating and managing quotes in SAP CPQ.



Standard Fields

Listed here are the standard fields that are present in quotes: quote header fields, quote item fields, product type fields, and total fields.
Quick Config (also referred to as Reverse Search )

The purpose of the Quick Config functionality is to provide a shortcut for searching and configuring the configurable product at the same time.
Quick Product Search in the Quote

Quick product search on ‘Add Products’ section is used for finding all types of product (simple or configurable) or favorites by typing any part of its part number, product name or description.
Scheduled Quote Deletion (Soft Delete)

Deleting quotes from the list of quotes only removes them from the user interface (doesn't delete them from the system) and marks them for a scheduled automatic deletion (soft delete).
Alerting Users About Parallel Work on Quotes (Administrator Side)

Quote 1.0 does not support parallel work on quotes. This means that only one user at a time can edit a quote.
