Create Products

Procedure
Go to Setup  Product Catalog  Products.

The list of all existing products displays. From this page, you can manually add new products, import them, or import the products' translations.

Click Add New Product.

The Definition tab is displayed.

Fill in the fields in the Definition tab to establish some general characteristics the product will have.
(Only if you use the Sales Area concept) Move on to the Sales Area tab to define specific characteristics that are relevant to each sales area.
Move on to the Pricing tab to define the pricing type that pertains to the product.
Click Save if you're creating a configurable product and you wish to edit other tabs as well.
Move on to the Attributes tab to define the product attributes.
Move on to the Rules and Messages tab if you wish to add rules which will control what can and can't be done during configuration.
Move on to the Scripts tab to add product-level scripts and specify events that trigger the execution of those scripts.
Move on to the Layout tab to change the position of product attributes in the Configurator and Responder, if needed.
Move on to the Advanced Settings tab to set up code matching.
Once you're done, click Save.

The product is created.
