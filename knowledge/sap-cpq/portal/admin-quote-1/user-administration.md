User Administration

The following topics guide the administrator through the Users section of the Setup.



Users

The Users administration page enables you to manage the existing users in the system and add new ones.
User Types

User types control how users interact with the system.
Permission Groups

The Permission Groups section in Setup  Users allows administrators to add new and modify existing permission groups.
User Custom Fields

User custom fields allow you to add additional fields for users.
Companies

Companies are organizations SAP CPQ users belong to.
Guest User

A guest user in SAP CPQ is a user with limited access rights. For example, guest users may need to log in only to browse products in the catalog with or without making the final purchase.
Access Rights

By managing access rights, you can control which individual administrators or permission groups can access specific Setup sections and entities, or grant them read-only access to those sections and entities.
