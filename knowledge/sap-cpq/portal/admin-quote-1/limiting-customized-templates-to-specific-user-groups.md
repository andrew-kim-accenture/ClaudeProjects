Limiting Customized Templates to Specific User Groups

You can limit the visibility of a customized responsive template to a specific user type.

Procedure
Go to Setup  User  User Types.

The page which contains all user types displays.

Click Edit next to the user type you wish to be able to see the applied customization.

The page in which you can edit the selected user type displays.

Click Add new custom template in the UI Design section.

A popup window displays.

Select the page, base template and the customized template you wish the user type to be able to see.
Click Add.
Click Save In the User Type page.
Results

Users who belong to the updated user type can now see the selected customized template in their environment. Other user types will see the template which is marked as Default. You can add more than one template here.
