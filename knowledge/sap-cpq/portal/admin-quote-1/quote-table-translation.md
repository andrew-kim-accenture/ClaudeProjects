Quote Table Translation

The Translations tab allows you to translate the Quote label, column and action labels and validation messages.

Each label has a dedicated row with a value in the default language and a value in the target language, if defined. To add a translation, first choose the target language in the Language drop-down list. You are now able to add the translated value in the Translation column.
