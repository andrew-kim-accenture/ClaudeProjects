Create Validation Rules

This topic contains the procedure for creating new validation rules.

Procedure
Click Add New Rule. 

The fields that need to be populated display in a popup. 

Enter the rule name in the Name field. 
Enter the message that will display if the rule is broken.
Use the Formula Builder to construct a formula in the Validation Formula field. 
Click Save Rule.
