Set Up Automatic Deletion of Data from the Audit Trail

You can set up automatic deletion of data from a specific Audit Trail log of your choice.

Procedure
Go to Setup  Security  Data Deletion.

A new page displays.

Under Audit Trail, find the audit trail for which you wish to enable automatic deletion and enable the dedicated Delete Audit Trail toggle switch.
Define a period after which the records from that particular audit trail will be automatically deleted.

The period can be set in either months or years.

Click Save.

A confirmation message displays.

All log entries older than the specified period are deleted daily.
