MRC Column in Existing Quotes

The purpose of this feature is to allow MRC total amount as a column in the list of existing quotes.

In Setup  Quotes  Column Headings administrator should add a new field ‘Total MRC Amount’ to the list of Available Fields.

This field will return total MRC amount on user’s side in the same way as the field ‘Total Amount’ returns total NRC price.
