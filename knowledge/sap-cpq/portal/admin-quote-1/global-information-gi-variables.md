Global Information (GI) Variables

Global Information (GI) variables contain values that can be read by any product or quote.

This comprehensive way of holding information allows values to be passed among quotes and products through the simple use of a tag. GI variables can be defined as a quote property, within a product model, or as a separate cart level aggregate. No matter where the GI was defined, it is available across all categories and products.



Create GI Variables

GI Variables can be created in two ways: explicitly and implicitly.
Retrieve GI Variables

Once a GI has been created, it is accessible from any category, product, or generated document.
