Favorites

Favorites reduce the time it takes to create quotes. They let you compile a list of favorite configurations, products, or services, which you can access with a single click.

This feature allows you to assemble a list of favorites, such as products sold most often to a specific customer or vertical.

To access the Favorites page, click Favorites in the Categories tree. The Favorites page lists previously added favorites. From here, you can configure, delete, edit, filter, or add favorites to a quote. Alternatively, you can access the Favorites page from the Configurator or a quote, by clicking the View Favorites icon in the Products actions.

Adding and Deleting Favorite Folders
Adding Folders

You can group favorites into folder structures for better overview and organization. New folders can be added under the root Favorites folder or as new subfolders. Folders can also be renamed.

You may use the same name for multiple folders at the same level within a folder tree, but it is not recommended, as it might lead to confusion.
Note

Favorite folders cannot be shared.

Deleting Folders

You can delete favorite folders by clicking the waste bin icon.

When you delete a folder, all the contents of the folder, including all subfolders, are deleted.

Favorite products and configurations you created are deleted.

Favorites shared by others aren't deleted. Instead, they're moved to the root Favorites folder.

Adding and Deleting Favorites

New favorites can be added in the following ways:

By clicking the Add To Favorites star icon above a product in the Catalog.

By clicking the Add to Favorites star icon in the Configurator, you can add the current configuration to the favorite folder.

The Add To Favorites popup window displays and you can enter the name and description of the configuration in the corresponding fields.

Furthermore, you can:

Add new folder by switching the toggle New Folder and defining the name of the folder. The folder will be added in the folder that is selected in the field Choose Location for New Folder. You can also add a favorite product or configuration to an existing folder by selecting the desired folder in the field Add to Existing Folder.

Tick the Share with users in my company checkbox if you want to share your favorites with other users in your company

Tick the Share with all users checkbox if you want to share your favorites with other users.

Once you click Save, the configuration appears in the Favorites page and the star icon changes its color to blue.

You can move shared favorites (shared by someone else) to a different folder.

One favorite can be only in one folder.

To delete a single favorite product or configuration, click on the blue star icon and confirm your choice

Note

The Share with users in my company and Share with all users options are displayed if the administrator has made them visible. The check-boxes are ticked by default, but you have the option to untick them if you don’t want to share the favorites with users in your company or all users in the system.

Viewing Favorites
You can access favorite configurations, products, or services:

from the Catalog, by clicking Favorites

from the Configurator, by clicking the View Favorites icon

from a quote, by clicking the View Favorites icon within Products actions

When you click the View Favorites icon in a quote or Configurator, you are redirected to the last visited favorites folder. Favorites and sub-folders within this folder are displayed, filtered by the product you're configuring. If you didn't visit any favorites folder, than you will be redirected to the root Favorites.

Note

Only favorites relevant to the corresponding product or quote item's product are shown, not all favorites in the folder. If none exist, no favorites are displayed.

When you open Favorites from the Catalog, no filtering by the product is executed.

Updating Favorites to a New Product Version
Note

This feature is only available if you're using the Quote 2.0 engine and your administrator has enabled it for you.

When a new product version is available, you will be prompted to update to the new version when you add a favorite to the quote, edit it, or configure it. Confirming this action ensures that your quote uses the latest product model.

Tip

Updating to the new version is highly recommended if you're working with favorites of variant configuration (VC) or advanced variant configuration (AVC) products. This way, your quotes always have the products synced with the latest active versions in SAP S/4HANA.

Shared Favorites
When a shared favorite has a new product version, both the owner and the user are prompted to update to the new version:

If the owner updates the favorite to the new product version, the user isn't prompted to update the favorite and uses the new version immediately.

If the owner doesn't update the favorite to the new product version, the user is prompted to update the product to the new version each time.
