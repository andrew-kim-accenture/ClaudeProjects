User Interface Design

This section contains useful information for managing user interface in SAP CPQ.

SAP CPQ uses a Responsive Design user interface. The Responsive Design allows users to customize most SAP CPQ pages using responsive templates for the desired page.

Responsive templates are the building blocks of pages in SAP CPQ’s Responsive Design. Responsive design templates define the look and feel of the application and allow users to customize page elemnets such as colors, icons, the order in which buttons are displayed and so on.

Most pages in the application consist of at least one template. For example, the Catalog page consists of the CatalogPage template, while the Approval History page consists of the ApprovalHistoryPage and Editors/DelegatedApprovers templates. Some templates are used on multiple pages in SAP CPQ. These templates are stored in the Shared page.
