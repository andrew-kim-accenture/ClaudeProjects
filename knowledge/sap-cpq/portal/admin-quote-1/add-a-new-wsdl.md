Add a New WSDL

This topic contains the procedure for adding WSDL files.

Context

If you’re adding WSDLs of web services with which you’ve previously established secure SOAP communication in Certificate Management, the service URL you provide when adding a WSDL file needs to be the same as the service URL you provide during the creation of a certificate.

Procedure
Go to Setup  Security  WSDL Management.

A page for managing and adding WSDL files displays.

Click Add New WSDL.

A popup window displays.

In the Service URL (required) field, enter the path to the endpoint of the secure gateway on which the WSDL file is installed.
Click Upload new file to add a WSDL.

The file should be in .xml format.

Click Save WSDL.
Results
Once a WSDL file is added to SAP CPQ, it will provide the means of communication between SAP CPQ and the server with which SOAP communication is performed.
