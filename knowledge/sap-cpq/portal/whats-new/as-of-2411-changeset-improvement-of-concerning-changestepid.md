As of 2411: ChangeSet Improvement of Concerning ChangeStepId

Description

If the client is using only the default ChangeSet (DEFAULT (CPQ-1)), the ChangeStepId will always be 0. In the case of any further custom ChangeSets being made, it will track changes with a ChangeStepId which can be from 0 to the maximum integer. If, in any case, the integer reaches its maximum value for ChangeStepId, it will be reset to 0 and start from the beginning.

Actions to Take

There is no need for any action.
