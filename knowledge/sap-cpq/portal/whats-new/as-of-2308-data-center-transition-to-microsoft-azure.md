As of 2308: Data Center Transition to Microsoft Azure

Description

In the next 2 to 6 months, most of the SAP CPQ cloud services will transition to the Microsoft Azure infrastructure. Solutions serviced in the SAP data centers as well as Disaster Recovery are planned to move to Microsoft Azure data centers.

Actions to Take

For a smooth transition, there are a few steps you have to take in preparation. For more information, see the Customer Checklist on the SAP CPQ Help Portal.
