As of 2311: Restricted File Types for Uploading into SAP CPQ

Description

To make the file uploading feature more secure, the file types dll, exe, asa, config, asp, com, aspx, cer, svg, xlsm, docm, pptm, db, tmp, log, url, rtf, html, htm are restricted to upload into SAP CPQ system.

Actions to Take

Customer might not be able to upload the restricted file types. Please try to use the supported formats for file uploading.
