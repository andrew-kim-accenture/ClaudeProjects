As of 2602: Removal of the Option to Switch to the Old UI Design

Description

The old and deprecated user interface design, Classic Design will be obsolete by the end of 2025. SAP CPQ recommends that clients using this old design switch to the standard Responsive Design, which offers many more advantages, such as UI customization of most of the SAP CPQ pages. All users will be switched to the Responsive Design with no possibility to switch back on Classic UI Design.

Actions to Take

Clients using the Classic Design must switch to Responsive Design by December 31st, 2025. If they need support from the SAP side during migration, they must reach out to SAP Support Team via a regular ticket.

To check if any users are using Classic Design, admins can go to Setup  General  Application Parameters  General Parameters .
