[CANCELED] As of 2511: Deprecation of CPQ Legacy SMTP Service and DKIM

Description
Note

This change was never implemented. The document is provided here for information purposes.

Starting with the November 2025 release (2511), SAP CPQ will deprecate the use of SAP CPQ Legacy SMTP service within SAP CPQ Email Settings, along with the associated DKIM support for outgoing emails. Effective from the 2511 release, any Email Settings that still utilize the SAP CPQ Legacy SMTP type will be automatically switched to the default SAP SMTP service (default), as the Legacy option will no longer be available in SAP CPQ.

Actions to Take
To switch from using the SAP CPQ Legacy SMTP type of service in the Email Settings in SAP CPQ Setup to one of the other two solutions:

SAP SMTP service (default)

Custom SMTP service
