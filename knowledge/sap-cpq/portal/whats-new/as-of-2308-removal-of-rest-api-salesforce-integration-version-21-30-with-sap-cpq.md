As of 2308: Removal of REST API Salesforce Integration (Version 21-30) with SAP CPQ

Description

As previously announced in the SAP CPQ 2211 Breaking Changes document, the Salesforce REST APIs version 21 to 30 are being retired. This means that the integration of SAP CPQ using these Salesforce APIs is now also retired with the SAP CPQ release 2308. The integration has moved to a partner-based integration.
