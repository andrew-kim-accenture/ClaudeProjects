As of 2511: Deprecation of PDA Approval Logic

Description

Users can approve quotes using the PDA link until the 2511 release. After the 2511 release, this functionality will be deprecated.

Actions to Take

As PDA approval is being deprecated, SAP CPQ requests that all customers using this feature migrate to One Click Approval in order to obtain approval links. For more information see, Approving Quotes in One Click.
