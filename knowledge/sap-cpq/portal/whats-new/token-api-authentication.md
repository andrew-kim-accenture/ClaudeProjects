Token API Authentication

The authentication types described in this section should be used for setup REST APIs. They all generate the same access token and it's up to the administrator to decide which authentication type to use. Additionally, these authentication types can also be used for SOAP APIs.

OAuth 2.0 Authorization Code Grant

The OAuth 2.0 authorization code can be used to provide individual users with tokens, so that there is no need to store the user's credentials in another application.
OAuth 2.0 Client Credentials Authentication

The OAuth 2.0 Client Credentials can be used to authenticate data between applications without the need for a specific user.
