As of 2408: Monthly Fields Relocated to General Settings Page from BRIM Set Up Page

Description
The placement of the The actual number of days in months and years and Fixed number of days fields has changed from the 2411 release. Since these fields are common and independent of BRIM integration, they have been moved from the BRIM providers page to the Mandatory settings page. The functionalities remain the same.
Actions to Take

Be aware of this placement change. If you have any automation running on these settings, it will be affected. Please make the necessary adjustments.
