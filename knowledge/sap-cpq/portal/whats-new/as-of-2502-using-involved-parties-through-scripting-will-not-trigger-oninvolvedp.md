As of 2502: Using Involved Parties Through Scripting Will Not Trigger OnInvolvedPartiesChange

Description

In order to be consistent with other event types, using InvolvedParties from scripting will not trigger the OnInvolvedPartiesChanged event from now on. Additionally, all related scripts attached to that event will not be executed in this flow.

Actions to Take

For 2502, this behavior will be controlled by a parameter, and customers will have one release, until 2505, to adjust their logic if needed.
