As of 2502: Removing Fields for Security Reasons

Description

Due to potential security issue on Business partners and Involved parties SAP CPQ removing fields under Additional information, that include Vat number, Tax number 1, Tax number2, Tax jurisdiction, Bank account 1, and Bank account 2.

Actions to Take
If any of the fields are in use, create custom fields with appropriate visibility permissions to replace the removed fields and contact SAP CPQ support for data migration. If necessary, modify your model (scripts, Iflows, formulas, and so on) to use the custom fields instead of the removed ones.
