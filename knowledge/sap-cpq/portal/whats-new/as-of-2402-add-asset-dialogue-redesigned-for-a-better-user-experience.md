As of 2402: Add Asset Dialogue Redesigned for a Better User Experience

Description

The Add Asset dialogue is redesigned for a better user experience when integration between SAP CPQ and BRIM is turned on.

Asset dialog until 2311: The user will see the list of subscription contract details on the initial page and based on the selected subscription contract details, item details will be displayed on the next screen. If the user wants to select more than one subscription contract document, then the toggle Allow multiple contracts will be present.

Asset dialog from 2402 release onwards: The user will see a list of subscription contract details and an expand button will be introduced for each subscription contract document. On choosing this button, item details will be present on the same page. On selecting the expand button for each item, the historical records of that contract will be present along with the status. The Allow multiple contracts toggle will be removed.

Actions to Take

If any customizations are enabled for the Add Asset dialog, they have to be adjusted or incorporated in the new Add Asset dialogue which will be available from 2402 release.
