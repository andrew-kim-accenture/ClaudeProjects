# IMarket

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/9a3f5cfd-8107-8735-3fbf-2b6f3d476251.htm

## Overview

Represents a market in SAP CPQ scripting.

## Properties

| Name | Type | Description |
|------|------|-------------|
| CurrencyCode | String | Gets the currency code |
| CurrencyDescription | String | Gets the currency description |
| CurrencyRate | Decimal | Gets the currency quote |
| CurrencySign | String | Gets the currency sign |
| ForwardCurrencyRate | Decimal | Gets the forward currency rate |
| MarketCode | String | Gets the market code |
| MarketFactor | Decimal | Gets the market factor |
| MarketName | String | Gets the name of the market |
| Selected | Boolean | Gets a value indicating whether this IMarket is selected |
