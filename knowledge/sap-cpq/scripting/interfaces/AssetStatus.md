# AssetStatus

**Namespace:** Scripting
**Type:** Enumeration
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/08ab5662-518e-3fab-d5fc-a59a9bada485.htm

## Overview

Defines different statuses for assets within the SAP CPQ system.

## Members

| Member Name | Value | Description |
|-------------|-------|-------------|
| Purchased | 0 | Represents a purchased asset |
| Obsolete | 1 | Represents an obsolete asset |
| Pending | 2 | Represents a pending asset |
| Renewed | 3 | Represents a renewed asset |
