# IReadOnlyProductMessages

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/03af3e28-be90-13be-ba4e-aea9e818b767.htm

## Overview

Product configuration template messages. Messages are shown in the responder part of the configurator.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Count | Int32 | Gets the count |
| Item | String | Gets the message at the specified index |

## Methods

### Contains(String message)
- **Returns:** Boolean
- **Description:** Determines whether message is already added to the product messages.
