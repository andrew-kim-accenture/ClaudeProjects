Provides information about a product exposed on 'On Configure' event.

| Name | Type | Description |
|------|------|-------------|
| OrderItemType | String | Gets Product Order Item type |
| PartNumber | String | Gets Product Part Number |
| SystemId | String | Gets Product System Id |
| UnitOfMeasure | String | Gets Product Unit Of Measure |
