# GenDocFormat

**Namespace:** Scripting
**Type:** Enumeration
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/3c807ade-62fc-cd97-30ce-ea2b9962696f.htm

## Overview

Represents output document formats used in the Generate Document quote action within SAP CPQ scripting.

## Members

| Member Name | Value | Description |
|-------------|-------|-------------|
| PDF | 0 | The PDF file format |
| DOC | 1 | The Word DOC file format |
| DOCX | 2 | The Word DOCX file format |
| EXCEL | 3 | The EXCEL file format. Note that source template file needs to be in EXCEL format as well; otherwise it won't work |
