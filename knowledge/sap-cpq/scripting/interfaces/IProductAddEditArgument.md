## Overview

Product added/updated argument is passed when the product added/updated to the quote event occurs.

## Properties

| Name | Type | Description |
|------|------|-------------|
| IsNewConfiguration | Boolean | Gets the value indicating if the product is added or updated to the quote |
| QuoteItemCollection | IQuoteItemCollection | Gets the collection of newly added quote items |
