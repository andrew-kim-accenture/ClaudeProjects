# IProductBeforeAddToQuoteEventArgument

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/8fa6170d-5ba3-bbee-4ffe-b0361b16541d.htm

## Overview

Before product Add to quote argument is passed before 'Product is added to the quote' event occurs.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Message | String | Gets or sets the message text of the product before add to quote validation message |
| Ok | Boolean | Gets or sets the value indicating if the product before add to quote validation is successful |
