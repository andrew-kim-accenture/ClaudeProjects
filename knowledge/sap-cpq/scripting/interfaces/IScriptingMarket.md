# IScriptingMarket

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/67c3c2d5-78cd-d6b3-fe8c-92ae449115a6.htm

## Overview

Provides access to market-related properties in SAP CPQ scripting, including currency and market configuration details.

## Properties

| Name | Type | Description |
|------|------|-------------|
| CurrencyCode | String | Gets currency code of market |
| CurrencyDescription | String | Gets currency description of market |
| CurrencyRate | Decimal | Gets currency rate of market |
| CurrencySign | String | Gets currency sign of market |
| IsDefaultMarket | Boolean | Gets whether this is the default market |
| MarketCode | String | Gets market code |
| MarketFactor | Decimal | Gets market factor |
| MarketId | Int32 | Gets market ID |
| MarketName | String | Gets market name |
| ReverseFactor | Decimal | Gets reverse factor of market |
| SystemId | String | Gets system ID of market |
| TimeZone | String | Gets time zone of market |
