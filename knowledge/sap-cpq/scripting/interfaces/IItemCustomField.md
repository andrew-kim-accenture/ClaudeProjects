# IItemCustomField

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/89ad8674-477d-7391-b6bf-023ca941e25d.htm

## Overview

Represents item custom field data. Contains additional data related to quote item custom field.

## Properties

| Name | Type | Description |
|------|------|-------------|
| CalculationFormula | String | Gets item custom field's calculation formula |
| CalculationOrder | Int32 | Gets item custom field's calculation order |
| DisplaySummary | Boolean | Flag indicating whether summary should be displayed |
| Id | Int32 | Gets item custom field id |
| Name | String | Gets item custom field's name |
| TypeId | Int32 | Gets item custom field's type id |
| TypeName | String | Gets item custom field's type name |
