# AttributeAccess

**Namespace:** Scripting
**Type:** Enumeration
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/f98846e7-b0f7-bd94-3ba2-87fd4f669886.htm

## Overview

Defines product attribute access restriction levels within SAP CPQ scripting.

## Members

| Member Name | Value | Description |
|-------------|-------|-------------|
| Editable | 0 | The attribute is editable |
| Hidden | 1 | The attribute is hidden |
| ReadOnly | 2 | The attribute is read only |
