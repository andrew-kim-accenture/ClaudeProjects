# IQuoteItemContainerRow

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/0e84fbc8-1cb7-399b-846f-f0ce244a9a90.htm

## Overview

Represents row of quote item's container attribute.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Columns | IQuoteItemContainerColumnCollection | Gets the columns of container row |
| IsSelected | Boolean | Gets value indicating if container row is selected |
| Item | String | Gets the value with the specified column name |
| RowIndex | Int32 | Gets container row index |
| UniqueIdentifier | String | Gets unique identifier |
