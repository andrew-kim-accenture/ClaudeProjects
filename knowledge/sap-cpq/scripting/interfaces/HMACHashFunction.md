# HMACHashFunction

**Namespace:** Scripting
**Type:** Enumeration
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/2703f402-c95e-167a-04b9-6dc62ad857d8.htm

## Overview

Represents different HMAC hash functions available in SAP CPQ scripting. Used for HMAC (Hash-based Message Authentication Code) operations.

## Members

| Member Name | Value | Description |
|-------------|-------|-------------|
| SHA256 | 0 | HMAC with SHA-256 hash function |
| SHA384 | 1 | HMAC with SHA-384 hash function |
| SHA512 | 2 | HMAC with SHA-512 hash function |
