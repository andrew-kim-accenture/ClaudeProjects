# ISqlParameter

**Namespace:** Scripting
**Type:** Interface
**Source:** https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/ef9540db-e6d2-248a-9ff8-c840a55bb30b.htm

## Overview

Represents a parameter within a SQL statement, facilitating the creation of parameterized SQL queries.

## Properties

| Name | Type | Description |
|------|------|-------------|
| DbType | SqlDbType | Gets or sets the SqlDbType of the parameter |
| ParameterName | String | Gets or sets the name of the ISqlParameter |
| Value | Object | Gets or sets the value of the parameter as an SQL type |
