# Scripting.Test Namespace

**Source:** `https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/178da83a-2e02-7a3e-166a-4fa15d095f0c.htm`
**Version:** 2603

## Overview

Interfaces for unit testing CPQ scripts. Provides container structures and product/attribute mocks for use in test scripts.

## Interfaces

| Interface | Description |
|-----------|-------------|
| `ITestContainer` | Test container — top-level container for test data |
| `ITestContainerCell` | Container cell (one row × one column) |
| `ITestContainerCellValue` | Value held in a container cell |
| `ITestContainerRow` | Container row |
| `ITestContainerRowCollection` | Collection of container rows |
| `ITestProduct` | Product information for testing |
| `ITestProductAttribute` | Product attribute for testing |
| `ITestProductAttributeValue` | Product attribute value for testing |
