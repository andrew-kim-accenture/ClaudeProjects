# Scripting.Quote.Approvals Namespace

**Source:** `https://help.sap.com/doc/6cbd2ad268f4444abbf41a52911c15e5/2603/en-US/Help/html/1e3f630a-3cbe-1fa3-ed33-200c80c7ea54.htm`
**Version:** 2603

## Overview

Approval workflow interfaces for Quote 2.0. All interfaces in this namespace are Quote 2.0 only.

## Interfaces

| Interface | Description |
|-----------|-------------|
| `IApprovalCondition` | Represents a broken approval rule |
| `IApprovalHistoryRecord` | Single record in the approval history of the quote |
| `IApprovalRequest` | Approval request sent to an approver |
| `IApprovalResponse` | Approver's response to an approval request |
| `IResponsibleApprover` | Responsible approver for broken approval rules |
| `ISubmitForApprovalRequest` | Request sent via the Submit for Approval action |
